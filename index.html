The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

Working mechanism of CSS box model
Box model has four parts- contain, padding, border and margin. First of all, it consider the weight and height as the contain of an element. Then it calculate the padding and then the thickness of border if there is any border and then it calculate the margin around the element!
